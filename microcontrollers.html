Teensy 3.1

I decided to get a Teensy after I accidentally killed the Metro Mini.
It is a huge step up in processing power, RAM and flash memory, and I/O.
So far, I have mostly been using it to drive an LCD but it has a lot of other capabilities.

More info on it can be found here

An overview of my current setup.


Metro Mini

The Metro Mini is essentially an Arduino Uno in a breadboardable formfactor. It afforded me more RAM and program storage than the picaxe, as well as some basic c++ skills, but sadly it died unexpectedly.

More info on it can be found here


Picaxe 20M2

For my first microcontroller, I wanted to start very simple. The Picaxe is a chip that comes with a custom bootloader, similar to arduinos, that allows it to run code written in BASIC.
While it is a bit limited, it was a good learning experience.

More info on it can be found here
